Monitoring Software

Dancom Monitoring Software enables logical and physical battery storage site visualization with real-time performance data for each individual module and for the whole system. It provides immediate fault detection and troubleshooting, efficient maintenance management, interactive charts and site layout that make it easy for users to ensure that the system is functioning properly.

Management Cloud

Dancom Cloud Based monitoring platform provides battery monitoring system with enhanced performance metering and controls. Data is sent securely to the cloud, where pre-made and customizable reports, dashboards, and visualizations are available via a secured web gateway. Fault-detection, monitoring, maintenance and many other features are included in DCB.

Embedded

Embedded systems consist of electronic circuits, analog devices, digital components, and intelligent controllers designed to work together.
